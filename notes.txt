Python is a high-level, interpreted programming language known for its simplicity, readability, and versatility. It was created by Guido van Rossum and first released in 1991.

Object-Oriented
Python supports object-oriented programming (OOP) concepts like classes and inheritance.

Python has thousands of libraries (like pandas, numpy, matplotlib, etc.) that make tasks easier.
